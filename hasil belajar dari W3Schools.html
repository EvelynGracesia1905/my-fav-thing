APA ITU HTML

HTML adalah singkatan dari HyperText Markup Language
html adalah bahasa markup standar untuk membuat halaman web
menggambarkan struktur halaman web
html terdiri dari serangkian elemen
Elemen HTML memberi tahu browser cara menampilkan konten
Elemen HTML memberi label pada bagian konten seperti "ini adalah judul", "ini adalah paragraf", "ini adalah tautan", dan lain-lain

Semua elemen HTML memiliki atribut yang menyediakan informasi tambahan tentang elemen.
Atribut selalu ditentukan di tag awal dan biasanya datang dalam pasangan nama seperti nama="nilai"
